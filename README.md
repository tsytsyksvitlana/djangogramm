# DjangoGramm

## Running aplication

1. Create and activate a virtual environment:

```
python -m venv myenv
```

On Windows:

```
myenv\Scripts\Activate
```

On Linux\MacOS:

```
source myenv/bin/activate
```

## API using

generate access token on http://127.0.0.1:8000/api/token/

enter Bearer {{your_access_token}} to header of request
